# Customer Churn Prediction using Artificial Neural Network

## Project Overview
This project applies an **Artificial Neural Network (ANN)** to predict whether a bank customer is likely to leave the bank or remain with it.

The objective of this project was to:
- preprocess customer data
- perform exploratory data analysis
- build a neural network model
- evaluate model performance

---

## Dataset
The dataset used in this project is the **Credit Card Customer Churn Dataset** from Kaggle.

### Target Variable
- **0** → Customer stayed
- **1** → Customer left

### Features
- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Active Member
- Estimated Salary

## Model Architecture
The ANN model contains:
- Input layer
- Hidden layer with 16 neurons
- Hidden layer with 8 neurons
- Output layer with 1 neuron

### Activation Functions
- ReLU
- Sigmoid

---

## Model Performance
The model achieved:

- **Accuracy:** 85.6%
- Precision
- Recall
- F1 Score
- Confusion Matrix
